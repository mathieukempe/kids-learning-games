# Game Ideas — Kids Learning Games

Researched game concepts based on learning theory (spaced repetition, retrieval practice, CRA progression, interleaving, ZPD scaffolding). Two daughters: 10yo and 8yo, bilingual English/French.

## Already Built

- **Multiplication Drop** (10yo) — Arcade falling blocks, tables 1-10
- **Multiplication Garden** (10yo) — Spaced repetition grid, adaptive difficulty
- **Dragon Days** (8yo) — Days of the week, EN/FR, 3 game modes

## Priority 1 — Building Now

- **Clock Tower Rescue** (8yo) — Telling time: climb a tower by reading analog clocks. Hours → half-hours → quarter-hours → 5-min intervals. Wrong = slip down.
- **Map Explorer** (Both) — Geography EN/FR: travel the world, identify continents/countries/capitals, earn passport stamps. Adaptive difficulty by age.
- **Conjugation Catapult** (Both) — French verbs: type conjugated form to launch boulders at a castle. Regular → irregular. Spaced repetition on weak verbs.
- **Potion Mixer** (8yo) — Addition/subtraction: select ingredients to hit a target number. Mix add/subtract. Visual cauldron theme.
- **Equation Quest** (10yo) — Pre-algebra: dungeon crawler with equations containing missing numbers. Mix operations. Three difficulty paths.
- **Number Line Ninja** (Both) — Number sense: estimate position on a number line. Ranges grow 0-10 → 0-100 → 0-1000. Star scoring by accuracy.

## Priority 2 — Future

- **Fraction Forge** (10yo) — Visual fractions, naming, comparing, adding. Blacksmith theme. CRA progression.
- **Word Bridge** (Both) — EN/FR vocabulary: type translations to repair a bridge. Picture hints. Spaced repetition.
- **Speed Division** (10yo) — Inverse of Multiplication Drop. Connects division to known multiplication facts.
- **Season Spinner** (8yo) — Months and seasons EN/FR. Spinning wheel. Extension of Dragon Days.
- **Pattern Lab** (Both) — Visual and number sequences. Science lab theme. Adaptive by age.

## Priority 3 — Nice to Have

- **Shape Shifter** (8yo) — 2D geometry: sort falling shapes into category bins.
- **Data Detective** (10yo) — Reading charts/graphs: detective cases with data clues.
- **Measurement Kitchen** (Both) — Units and conversion through cooking recipes.
- **Spell Cascade** (Both) — Spelling EN/FR: type words before letters fall.

## Learning Principles Applied

- **Spaced repetition** — weak items resurface more often
- **Retrieval practice** — typing > multiple choice for memory
- **CRA** — concrete visuals before abstract notation
- **Interleaving** — mix problem types within sessions
- **ZPD** — adaptive difficulty with scaffolding
- **Bilingual reinforcement** — language toggle leverages cognitive flexibility
